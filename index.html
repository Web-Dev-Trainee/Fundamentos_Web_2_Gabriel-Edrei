Parte 1: Questões

Questão 01: Quais são os oito tipos de dados de Javascript?

Os oito tipos de dados de Javascript são Number, Null, String, Boolean, BigInt, Undefined, Symbol e Object. Veja um exemplo de cada um deles na imagem ao lado.

Questão 02: Qual tipo de dados não é primitivo?

Tipos de dados como Vetores e matrizes, não são considerados primitivos, pois são estruturas bem mais complexas.

Questão 03: Qual é a diferença entre aspas simples, duplas e crase para strings?

No Javascript, não existe diferença entre aspas duplas e simples, ambas possuem a mesma finalidade, porém, são intercâmbiadas para serem usadas uma dentro da outra. Já as crases são usadas para concatenar em Template String

Questão 04: Que tipo de citação permite incorporar variáveis ​​/ expressões em uma string?

A citação que permite tal ação é chamada de Template String na 5ª versão do ECMAScript, e de Template Literals a partir da ES6.
