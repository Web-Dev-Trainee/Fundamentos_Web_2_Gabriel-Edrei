Parte 1: Questões

Questão 01: Quais são os oito tipos de dados de Javascript?

Os oito tipos de dados de Javascript são Number, Null, String, Boolean, BigInt, Undefined, Symbol e Object. Veja um exemplo de cada um deles na imagem ao lado.

Questão 02: Qual tipo de dados não é primitivo?

Tipos de dados como Vetores e matrizes, não são considerados primitivos, pois são estruturas bem mais complexas.

Questão 03: Qual é a diferença entre aspas simples, duplas e crase para strings?

No Javascript, não existe diferença entre aspas duplas e simples, ambas possuem a mesma finalidade, porém, são intercâmbiadas para serem usadas uma dentro da outra. Já as crases são usadas para concatenar em Template String

Questão 04: Que tipo de citação permite incorporar variáveis ​​/ expressões em uma string?

A citação que permite tal ação é chamada de Template String na 5ª versão do ECMAScript, e de Template Literals a partir da ES6.

Questão 05: Como você incorpora variáveis/expressões em uma string?

Por meio de Template String, usando os operadores concatenados.

Questão 06: Como você evita caracteres em uma string?

Usando o método substr(), que retorna somente os índices indicados.

Questão 07: Qual a diferença entre slice/substring/substr?

O método slice() retorna uma cópia de um pedaço do array a partir de um subarray criado entre o início e o fim do índice do mesmo. O array de origem não sofre alteração.
O método substring() retorna uma parte da string entre os índices inicial e final.
O método substr() retorna uma parte uma parte da string a partir de um índice especificado e se estendendo até um número de caracteres que também é especificado.

Questão 08: O que são métodos?

Métodos são funções que são invocadas por objetos.

Questão 09: Quais são os 3 operadores lógicos e o que eles representam?

Os 3 operadores lógicos são:
(!) NOT = Negação. pode ser usado em qualquer tipo de valor.
(||) OR = Usado para retornar um valor entre dois valores comparados.
(&&) AND = Retorna um valor quando os dois valores comparados cumprem o requisitado.

Parte 2: Exercícios

Exercício 1

A função no exercício deveria estar registrando o número 2, mas não está retornando esse valor. A solução para o problema foi retirar as aspas que estavam nos números dentro do return, e o problema foi resolvido.

Exercício 2

Na sintaxe deste exercício, o navegador solicita um número para o usuário, e caso esse número seja 6, é retornado um valor true, caso contrário, um valor false é retornado. O exercício pede que seja feita uma forma de retornar um valor verdadeiro quando o número digitado for maior ou igual a 10, e um resultado falso quando o número for menor que 10.
A solução para o exercício foi alterar os operadores de "===" para ">=" e o número de 6 para 10
